# Welcome Note

This is a sample vault note for vault-agent testing and demonstration.

## Overview

Vault-agent is a standalone search and retrieval server for Markdown vaults. It provides:

- Lexical search using FTS5
- Optional embedding-based semantic search
- Hybrid search combining both signals
- Related note discovery
- Explicit note and chunk retrieval

## Getting Started

1. Configure your vault root path
2. Start the server with `vault-agent serve`
3. Search with `vault-agent search "your query"`
4. Retrieve notes with `vault-agent get note <note-id>`
